# Comsats University Islamabad - Job Application Form

A web-based Non-Faculty Job Application form for Comsats University Islamabad. This project utilizes HTML and CSS to create a structured and responsive user interface for applicants to fill out their details.

## Features
- **University Branding:** Includes the official Comsats University Islamabad logo.
- **Application Details Section:** Collects information like Post applied for, Department, and Campus (Sahiwal or Attock).
- **Personal Information:** Captures the applicant's Full Name and Father's Name.
- **Photo Placeholder:** Dedicated area for applicants to attach their 5cm x 5cm photograph.

## Tech Stack
- **HTML:** Structured content and form fields.
- **CSS:** Styling the application layout, colors, and typography.

## Setup Instructions
1. Clone this repository to your local machine.
2. Open the `index.html` file in any modern web browser to view the application form.
